## JFDF介绍
>JFD的全称是JAVA Fast Development Framework，即JAVA快速开发框架。<br>
* **JFDF快速构建应用系统** >JFDF可以帮助大家快速投入到web项目的开发当中，从而避免了许多程序开发初期阶段不必要的麻烦，诸如各种开源技术的整合、开发规范、项目结构、访问权限控制、用户角色管理、系统菜单管理、必要数据的初始化等等。相信每一个独立研发过产品的攻城狮来说，都经历过以上的这些工作，繁琐、枯燥、乏味，每天伴随着的是数不清的调试和问题以及有兼容性错误，久而久之让我们不得不怀疑人生，常常认为自己已然成为了一个问题青年。<br>
总之，JFDF建立的初衷是想能够帮助你迅速构建起你的业务系统，让每一位开发人员的视角聚焦在业务功能上，从而避免不必要的麻烦。
